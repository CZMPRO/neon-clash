# 零界武斗：NEON CLASH

原创离线像素格斗游戏。直接双击 `index.html` 即可运行，无需安装依赖。

## 游戏模式

- **快速对战**：三局两胜，支持简单、普通、困难三档公平 AI。
- **零界塔**：连续五层生存挑战，每层胜利后从三项强化中选择一项，第五层为强化 Boss。
- **实战教学**：逐步练习移动、轻击、重击、特殊技、防御与终结技。

进度、辅助设置、零界塔最高层数与最高分保存在浏览器本地。

## 操作

- A / D：移动
- W：跳跃
- S：低姿态
- Shift + 方向或双击方向：冲刺 / 撤步
- J / K / L：轻击 / 重击 / 特殊技
- I：防御与完美防御
- U：满能量终结技
- Esc：暂停

支持标准手柄：摇杆移动，A 跳跃，X/Y/B 对应轻击/重击/特殊技，LB 防御，RB 终结技。

## 辅助功能

- 音乐、音效、屏幕震动独立调节
- 高对比 HUD
- 减弱闪光特效
- 页面失焦时自动暂停

建议使用最新版 Chrome、Edge 或 Firefox。
